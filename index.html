RubberSoul

Full-stack e-commerce site with React.

E-commerce site for rock climbing shoes. Allows customers to browse and search shoes by brand, model, and tags. Users, reviews, and inventory are stored in a RESTful backend database.

  • Date: March 2018
  • Client: RubberSoul
  • Category: Full-stack

Mossy Mood Tracker

Full-stack site with REST API and database.

Tracker to allow users to report interactions with Galvanize Seattle's data science dog, Mossy. Each submission is stored in a backend database, and interactions are displayed to help the user understand the best time of day to play with Mossy.

  • Date: February 2018
  • Client: Mossy
  • Category: Full-stack

Hogwarts House Ipsum

Interactive Sorting Hat & ipsum generator.

Hogwarts House Ipsum is a simple Hogwarts house sorting hat and ipsum generator. It supports generating text in one, two, or three sentence lengths, with different outputs based on the Hogwarts house selected.

  • Date: December 2017
  • Client: Hogwarts
  • Category: UX/UI & Website Design
